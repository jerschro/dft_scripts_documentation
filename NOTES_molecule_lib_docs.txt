how to upload to github and create website
git init -b main (in powershell in folder)
in github desktop
Click on current repository
Click Add - Add Existing repository
Commit (bottom left)

Now generate mkdocs-ghdeploy

How to update website:
update git
go to github.com/repo settings - pages - (next to visit site) 3 dots - unpublish site
python3 -m mkdocs build
python3 -m mkdocs gh-deploy
go to actions in repo
watch website get built
navigate to updated website
